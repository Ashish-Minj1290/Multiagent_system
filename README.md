## Overview

**ORAEON** is an advanced Multi-Agent Research Intelligence System designed to automate deep research workflows using a coordinated network of AI agents.

The platform transforms a simple user query into a comprehensive intelligence report by orchestrating multiple specialized agents that search, analyze, synthesize, and critique information in real time.

Instead of relying on a single AI response, ORAEON follows a structured research pipeline where each agent performs a dedicated task, resulting in higher-quality, more reliable, and better-organized outputs.

## Agent Responsibilities

### 1. Search Agent

Responsible for discovering relevant information from various sources.

**Tasks:**

* Web search
* Source collection
* Information retrieval
* Data gathering

---

### 2. Reader Agent

Processes and analyzes the collected information.

**Tasks:**

* Content extraction
* Source understanding
* Key insight identification
* Information filtering

---

### 3. Writer Chain

Transforms extracted insights into a coherent research report.

**Tasks:**

* Report generation
* Knowledge synthesis
* Structured writing
* Summary creation

---

### 4. Critic Chain

Reviews and improves the generated output.

**Tasks:**

* Fact consistency checking
* Quality assurance
* Gap identification
* Report refinement

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/oraeon.git

cd oraeon
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

#### Windows

```bash
.venv\Scripts\activate
```

#### Linux / Mac

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
